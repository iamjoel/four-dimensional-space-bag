# 元素跟随(Portal to follow elem)
基于：[floating-ui](https://floating-ui.com) 做的封装。
可以做 Tooltip。功能：
1. 控制弹出内容的方向。上下左右，以及侧轴的开始，居中，结束。
2. 弹出内容是 Portal 到 body 下的。
2. 滚动时弹出内容会跟随。
3. 弹出内容在栋页面不显示的地方时，会翻转到反方向。
