# 布局
## 概览
布局：位置 + 大小。

### 布局基础
* 盒模型。margin，content，padding。
* 渲染模式。标准模式，怪癖模式。

### 布局技术
* Flex
* Grid
* 定位布局。绝对定位。固定定位。sticky定位。
* 响应式布局。
* 列布局
