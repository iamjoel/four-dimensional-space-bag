# 工具 hooks

## usePortal
将某个元素移动到层级更浅的子元素上。  
**使用场景：** 子元素的部分内容想超出 overflow: hidden 或 带滚动条的父元素时，仍然显示全。

    {() => <UsePortalDemo />}
  </BrowserOnly>
</Demo>

      }}
    >

    </div>
  </Portal>
)`}
