### Excel 类表格
* Airtable
* 字节的多维表格。2023/11/8 通过 HTTP 来改记录或新增记录很不灵活:
  * 改一列的值。要么改一列的一样。要么用同一个值改所有行。
  * 只能新增一列。不支持新增很多列。
  * 从 HTTP 接口拿值。text 格式是值就是整个 HTTP 返回的(Header,body 等)。拿 JSON 格式的，只能拿整个 JSON 的值。
