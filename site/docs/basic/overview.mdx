# 一些基础的东东

## 决策
### 理性决策
#### [《理性决策的方法 | 5Y View》](https://mp.weixin.qq.com/s/YX6aoOOx77BDwt4n1hSqUw)
五源资本写的。文章内容来自：盖伊·斯皮尔的书《与巴菲特共进午餐时，我顿悟到的5个真理》。

问题：大脑大部分时间的思考是非理性的(《思考的快与慢》中的快速思考)。以经济为例，

经济是复杂的动态系统。标准的经济世界模型不能充分反映市场实际，用生物模型可能更好。（标准的经济世界模型具体指？经济模型过度简化了）。
生物模型是关于种族如何共同进化和互相竞争的。

如何避免自身的思维误区？
大脑是不完美的。有的人注意力缺乏症，容易忘东西。发现不完美，营造合适的环境去改造它。

#### 减少外界环境造成的分心。
> 心理学家罗伊·波美斯特曾说明，克制力是一种有限的资源，所以我们必须小心谨慎，不要把它用竭。

作者选择住在苏黎世。原因：
安静、舒适，没有太多的外界刺激。因为分心是一个现实问题。**我真正需要的是没有过度兴奋的、淳朴、不引人注目的背景环境。**而且，我肯定不是第一个发现苏黎世有益于清醒思考的人。在历史上，这座城市为自由思想提供了场所，居住过卡尔·荣格、詹姆斯·乔伊斯、理查德·瓦格纳、列宁以及阿尔伯特·爱因斯坦等人，蒂娜·特纳之类的就更不用说了。

办公室的特点：里边很少有东西会扰乱他的思维。
