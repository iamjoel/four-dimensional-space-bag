# python

## 将安装包保存到 requirements.txt
```
pip install package && pip freeze > requirements.txt
```

其中：`pip freeze` 命令将当前python 环境安装的软件包列表输出到终端。更好的方式，用 pipreqs 这个工具。
安装
```
pip install pipreqs
```

```
pipreqs ./ --encoding=utf8 --force
```

## 环境
### Mac 上把 python3 设置为默认的 python 版本
在 `~/.zsh_profile` 中添加
```bash
alias python=python3
alias pip=pip3
```
